# TAREA 2
Creación de un formulario usando funciones de JS para devolver mensaje en pantalla.

## Estructura del Proyecto
- `index.html` → Contiene la estructura de la página  
- `style.css` → Estilos  
- `script.js` → funciónes del formulario. 

## Instrucciones para clonar el repositorio
1. Descargar o clonar el repositorio 
git clone https://github.com/andreacastillo2022-gif/tarea2-utn.git
2. ingresar a la carpeta
    cd tarea2-utn
3. Ejecutar la tarea: requiere abrir el archivo index.html en cualquier navegador.
